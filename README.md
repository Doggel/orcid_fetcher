# getORCIDworks
Input ORCIDs and get the informations of their works, including title, journal, pulication date, doi, etc.

# How to Use
- Input 'Name' and 'ORCID' in excel file 'orcid_list.xlsx'. The format is as follows.
<img width="298" height="79" alt="image" src="https://github.com/user-attachments/assets/51b2f022-5717-4de6-a08a-096df5cd54fd" />

- Run the script getWorksByORCID.py
- Works will be export to excel file 'output_orcid_works.xlsx'.
